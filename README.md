# Playwright MCP Automation Framework

## 📌 Overview

This project demonstrates **End-to-End (E2E) Web Automation** of a banking application using **Playwright** and **JavaScript**.

It serves as a **Proof of Concept (POC)** for building a scalable and maintainable automation framework integrated with an MCP-based setup.

---

## 🚀 Key Features

* End-to-End test automation for banking workflows
* Built using **Playwright** with JavaScript
* Modular and scalable framework design
* Integration with **Playwright MCP Server**
* Smart code generation and assistance using **GitHub Copilot**
* Reusable utilities and test components

---

## 🛠️ Tech Stack

* **Playwright**
* **JavaScript (Node.js)**
* **Playwright MCP**
* **GitHub Copilot**

---

## 📂 Project Structure

```
├── tests/              # Test cases (E2E scenarios)
├── pages/              # Page Object Models
├── test-data/          # Test data for data driven testing
├── fixtures/           # Test setup and fixtures
├── config.json         # Configuration files
└── package.json        # Project dependencies
```

## ▶️ Getting Started

### Prerequisites

* Node.js (v16 or above recommended)
* npm or yarn installed

### Installation

```bash
npm install
```

### Run Tests

```bash
npx playwright test
```

### Run Tests in UI Mode

```bash
npx playwright test --ui
```

---

## 📊 Reporting

Playwright provides built-in reporting:

```bash
npx playwright show-report
```
